简介

快速上手

这个原因是系统找不到你的 vs shell 位置，原因可能是：

  • 没安装
  • 没安装正确

从这里下载好 vs shell

安装前删除注册表：计算机\HKEY_CURRENT_USER\Software\Microsoft\SQL Server Management Studio\14.0_Remote

FAQ
