###在CentOS 6.7下搭建DHCP服务

  • [~]# yum install dhcpd #安装dhcp 0X01 配置服务器网络服务

首先我们要将本地的网络配置成静态地址，并重启网络服务

  • [~]# vi /etc/sysconfig/network-scripts/icf-eth0 #将本地的网络配置成静态地址，并重启网络服务

  • [~]# service network restart
  • [~]# vi /etc/dhcp/dhcpd.conf
