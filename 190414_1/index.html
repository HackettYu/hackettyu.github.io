FAQ

  • 报 404: 先开启服务器

  • 报 403:

    无法进入 manager 页面

  1. 打开 http://localhost:8080/manager/html 能否用配置好的账号登录
  2. ${Tomcat8_HOME_PATH}/webapps 下有 manager 文件, 如果之前删过, 拷贝一份回来
  3. 确认 username 和 password 正确

就是账号没配置好, 回去确认
